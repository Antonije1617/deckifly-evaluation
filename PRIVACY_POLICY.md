# Deckifly Privacy Policy

## Overview
Deckifly is a browser extension designed to help users collect vocabulary while reading online and create flashcards in Anki.

## Data Collection
Deckifly does not collect personally identifiable information.

For research purposes, the extension may collect anonymized usage data, including:
- interactions with the extension (e.g. button clicks)
- number of saved flashcards
- usage of AI-generated definitions
- interactions with AI recommendations

## Website Content
The extension processes text content from webpages locally in the browser to enable vocabulary extraction and AI suggestions. This content is not stored permanently or linked to user identity.

## Data Usage
Collected data is used exclusively for:
- academic research purposes (Master’s thesis)
- improving the functionality of the extension

## Data Sharing
No personal data is sold, shared, or transferred to third parties.

## Backend Communication
The extension communicates with a backend service to generate AI-based vocabulary suggestions and definitions. Only necessary text data is sent for processing.

## Storage
User preferences (e.g. selected Anki deck, settings) are stored locally in the browser using Chrome storage.
